Returns true if self contains the given field.

Note: If field shares a name with a field in this FieldSet, but was created by a FieldSet with a different callsite, this FieldSet does not contain it. This is so that if two separate span callsites define a field named "foo", the Field corresponding to "foo" for each of those callsites are not equivalent.
